## 第二步：用turtle画爱心

turtle是Python自带的"小海龟"，它就像一支会听你指挥的画笔🐢

!!! example 代码1：画个爱心送给你
    新建文件 love.py，输入：

```py
import turtle
import time

# 设置画布
screen = turtle.Screen()
screen.title("💖 送给你的爱心")
screen.bgcolor("white")

# 创建画笔
pen = turtle.Turtle()
pen.shape("turtle")
pen.color("red")
pen.speed(3)  # 速度：1-10，数字越大越快
pen.pensize(3)

# 画爱心的函数
def draw_heart():
    pen.begin_fill()
    pen.left(140)
    pen.forward(113)

    # 画左边的圆弧
    for i in range(200):
        pen.right(1)
        pen.forward(1)

    pen.left(120)

    # 画右边的圆弧
    for i in range(200):
        pen.right(1)
        pen.forward(1)

    pen.forward(113)
    pen.end_fill()

# 开始画画
pen.penup()
pen.goto(0, -100)
pen.pendown()

draw_heart()

# 写上文字
pen.penup()
pen.goto(0, 50)
pen.color("black")
pen.write("I Love Python!", align="center", font=("Arial", 20, "bold"))
pen.hideturtle()

# 点击关闭
screen.exitonclick()
```

!!! info 运行效果：

    一只小海龟会慢慢画出一个红色爱心❤️
    中间写着 "I Love Python!"

## 第三步：用turtle画彩色星星

!!! example 代码2：绘制彩虹星星
    新建文件 star.py：

```py
import turtle
import random

# 设置画布
screen = turtle.Screen()
screen.bgcolor("black")  # 黑色背景更炫！
screen.title("🌟 彩虹星星")

# 创建画笔
pen = turtle.Turtle()
pen.speed(0)  # 最快速度
pen.width(2)

# 彩虹颜色
colors = ["red", "orange", "yellow", "green", "blue", "purple", "pink"]

# 画星星的函数
def draw_star(size):
    for i in range(5):
        pen.color(colors[i % len(colors)])
        pen.forward(size)
        pen.right(144)

# 画多个星星
pen.penup()
pen.goto(-200, 100)
pen.pendown()

for i in range(10):
    size = random.randint(20, 100)
    x = random.randint(-300, 300)
    y = random.randint(-300, 300)

    pen.penup()
    pen.goto(x, y)
    pen.pendown()
    draw_star(size)

pen.hideturtle()
screen.exitonclick()
```

!!! info 运行效果：

    黑色背景上，随机出现10个彩色星星✨
    每次运行位置都不一样！

## 第四步：用matplotlib画3D绚丽花朵

现在我们来点更专业的！用matplotlib画出数学之美🌸

!!! example 代码3：数学公式生成的花朵

    新建文件 flower_3d.py：

```py

import numpy as np
from matplotlib import pyplot as plt
from mpl_toolkits.mplot3d import Axes3D
from matplotlib import cm
# 创建图形
fig = plt.figure(figsize=(10, 8))
ax = fig.add_subplot(111, projection='3d')
# 生成花朵数据
[x, t] = np.meshgrid(np.array(range(25)) / 24.0, 
                     np.arange(0, 575.5, 0.5) / 575 * 17 * np.pi - 2 * np.pi)
p = (np.pi / 2) * np.exp(-t / (8 * np.pi))
u = 1 - (1 - np.mod(3.6 * t, 2 * np.pi) / np.pi) ** 4 / 2
y = 2 * (x ** 2 - x) ** 2 * np.sin(p)
r = u * (x * np.sin(p) + y * np.cos(p))
h = u * (x * np.cos(p) - y * np.sin(p))
# 绘制3D表面
surf = ax.plot_surface(r * np.cos(t), 
                       r * np.sin(t), 
                       h, 
                       rstride=1, 
                       cstride=1, 
                       cmap=cm.cool,  # 使用冷色调
                       linewidth=0, 
                       antialiased=True,
                       alpha=0.8)
# 设置
ax.set_box_aspect([1,1,1])
plt.axis('off')
plt.title('3D Flower', fontsize=15, pad=20)
plt.show()
```

!!! info 运行效果：
    带动画的旋转花朵，保存为动图（超炫！）

!!! example 代码6：旋转的3D花朵（保存GIF）
    新建文件 flower_3d_gif.py：

```py
import numpy as np
from matplotlib import pyplot as plt
from matplotlib import animation
from mpl_toolkits.mplot3d import Axes3D
from matplotlib import cm

# 创建图形
fig = plt.figure(figsize=(10, 8))
ax = fig.add_subplot(111, projection='3d')

# 生成花朵数据
[x, t] = np.meshgrid(np.array(range(25)) / 24.0, 
                     np.arange(0, 575.5, 0.5) / 575 * 17 * np.pi - 2 * np.pi)
p = (np.pi / 2) * np.exp(-t / (8 * np.pi))
u = 1 - (1 - np.mod(3.6 * t, 2 * np.pi) / np.pi) ** 4 / 2
y = 2 * (x ** 2 - x) ** 2 * np.sin(p)
r = u * (x * np.sin(p) + y * np.cos(p))
h = u * (x * np.cos(p) - y * np.sin(p))

# 初始化函数
def init():
    ax.plot_surface(r * np.cos(t), r * np.sin(t), h,
                   rstride=1, cstride=1, cmap=cm.gist_rainbow_r,
                   linewidth=0, antialiased=True, alpha=0.9)
    plt.axis('off')
    return fig,

# 动画函数
def animate(i):
    ax.view_init(elev=20, azim=i*2)  # 旋转视角
    return fig,

# 创建动画
ani = animation.FuncAnimation(fig, animate, init_func=init,
                              frames=180, interval=50, blit=False)

# 保存为 GIF
print("正在保存动画为 GIF，请稍候...")
ani.save('3d_flower.gif', writer='pillow', fps=30)
print("保存完成！文件名为：3d_flower.gif")

# 显示动画
plt.show()
```

!!! info 运行后：

    当前目录会生成一个 flower_3d.gif 文件
    花朵会360度旋转展示
    可以分享给朋友炫耀啦！😎


## Q & A

!!! question Q1：运行时报错"ModuleNotFoundError"怎么办？

    A：说明某个库没安装好，重新运行：

    `pip install matplotlib numpy`

!!! question Q2：中文显示成方框怎么办？

    A：尝试更换字体，把代码中的：

    `plt.rcParams['font.sans-serif'] = ['SimHei']`

!!! question Q3：3D图旋转卡顿怎么办？

    A：减少网格密度，把 `range(25)` 改成 `range(15)`，会更快但没那么精细

!!! question Q4：保存GIF失败怎么办？

    A：确保安装了pillow：

    `pip install pillow`
